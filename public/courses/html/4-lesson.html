Урок 4: Работа с изображениями и мультимедиа

В этом уроке вы изучите, как вставлять изображения, аудио и видео в HTML, а также узнаете о настройке размеров, использовании альтернативного текста и создании встроенных плееров для медиа-контента.

Теория

HTML предоставляет ряд элементов для работы с изображениями и мультимедиа. Ниже приведены основные теги и атрибуты, которые вам необходимо знать:

Изображения

  • <img> — используется для вставки изображений. Обязательный атрибут — src (путь к файлу изображения).
  • Атрибут alt задаёт альтернативный текст для изображения, что улучшает доступность и SEO.
  • Атрибуты width и height позволяют задать размеры изображения.

Аудио

  • <audio> — позволяет встроить аудио на страницу. Основной атрибут — src для указания пути к аудиофайлу.
  • Атрибут controls добавляет элементы управления воспроизведением (плей, пауза, регулировка громкости и т.д.).
  • Можно использовать вложенный тег <source> для указания нескольких форматов аудио.

Видео

  • <video> — используется для вставки видео. Задаётся с помощью атрибута src или вложенных тегов <source>.
  • Атрибут controls добавляет панель управления видео (кнопки воспроизведения, перемотки и т.д.).
  • Атрибуты width и height задают размеры плеера.

Используя эти элементы, вы можете создавать интерактивный мультимедийный контент на своих веб-страницах, делая их более информативными и привлекательными для пользователей.
