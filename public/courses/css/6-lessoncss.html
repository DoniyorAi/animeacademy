Урок 6: CSS Grid

В этом уроке ты научишься использовать CSS Grid для создания сложных, гибких макетов. Узнай, как разделять страницу на колонки и строки, задавать промежутки и управлять расположением элементов.

Документ Видео Задания

Теория

CSS Grid – это современный инструмент для создания макетов, который позволяет легко разделить страницу на строки и колонки. С его помощью можно задавать размеры, промежутки и области для размещения элементов.

Вы изучите, как использовать свойства grid-template-columns, grid-template-rows, gap и другие, чтобы создать удобную и адаптивную сетку.
