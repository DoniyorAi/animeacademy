Урок 5: Flexbox и макеты

В этом уроке мы изучим основы Flexbox – мощного инструмента для создания адаптивных макетов, выравнивания элементов и удобной верстки для любых экранов.

Документ Видео Задания

Теория

Flexbox позволяет легко управлять расположением элементов внутри контейнера. Чтобы создать flex-контейнер, достаточно задать для родительского элемента свойство display: flex.

Основные свойства, такие как justify-content и align-items, обеспечивают выравнивание по горизонтали и вертикали соответственно. Кроме того, свойство flex-wrap позволяет элементам переноситься на новую строку, что делает макеты адаптивными и удобными для любых экранов.
